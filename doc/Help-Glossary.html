MOSFET 輸出特性曲線 (ID vs VDS) 區域說明

1. 線性區 (Triode Region / Ohmic Region)

位置： 圖形的左側，當 VDS 較小時，曲線呈現近似直線的一段。

  • 物理特性： 元件行為類似一個 可變電阻。隨著 VDS 增加，電流 ID 也會線性增加。
  • 工作狀態： 這是 MOSFET 當作 開關導通 (ON) 時所在的區域。
電路上的意義：
在這個區域，斜率的倒數就是導通電阻 RDS(on)。如果 VDS 很小，曲線幾乎是一條直線。

2. 飽和區 (Saturation Region / Active Region)

位置： 圖形的右側，曲線逐漸趨於平坦的區域。

  • 物理特性： 元件行為類似一個 電流源。電流 ID 主要由閘極電壓控制，幾乎不受 VDS 影響。
  • 工作狀態： 這是 MOSFET 當作 放大器 時的主要工作區域。
電路上的意義：
若忽略通道長度調變 (λ ≈ 0)，則曲線為水平直線；若 λ > 0，則會隨 VDS 略為上升。

漏電流參數說明 (Leakage Current)

IDSS：漏極漏電流 (Zero Gate Voltage Drain Current)

物理意義： 當閘極電壓 VGS = 0V，且 MOSFET 處於 截止狀態 時，仍然存在的微小漏電流。

  • 測試條件： 通常在 VDS = 最大額定值 VDSS 時測量。
  • 數值範圍： 一般為 (µA) 等級。
電路上的意義：
因為特性曲線通常以安培為單位，IDSS 太小，在圖上幾乎貼近 0 軸，不容易看出。

IGSS：閘極漏電流 (Gate-Body Leakage Current)

物理意義： 通過閘極氧化層的極小電流，代表 絕緣層的漏電現象。

  • 測試條件： 在 VDS = 0V，且閘極施加最大額定電壓時測量。
  • 數值範圍： 通常為 (nA) 等級。
電路上的意義：
IGSS 屬於閘極電流，不屬於漏極電流 ID，因此 不會影響 ID-VDS 曲線。
注意：
MOSFET 的「飽和區」與 BJT 的名稱相反。在 BJT 中，飽和區代表開關導通，而放大區則稱為主動區，請勿混淆。

* 若 IDSS 或 IGSS 數值異常偏大，通常表示元件損壞、漏電或操作條件超出規格。
