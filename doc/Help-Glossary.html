MOSFET 輸出特性曲線 (ID vs VDS) 區域說明

1. 三極管區 (Triode Region / Ohmic Region)

位置： 位於圖表的最左邊，當 VDS 較小時，曲線呈現陡峭上升的那一段。

  • 物理特性： 電晶體此時表現如同一個「受控的變值電阻」。當 VDS 增加，電流 ID 也會隨之快速增加。
  • 應用狀態： 這是電晶體作為「開關」且處於導通 (ON) 狀態時所在的區域。
在本程式中的表現：
此區域的斜率倒數即為導通電阻 RDS(on)。如果 VDS 極小，曲線幾乎呈直線。

2. 飽和區 (Saturation Region / Active Region)

位置： 位於圖表中間偏右側，曲線呈現水平、平坦的大片區域。

  • 物理特性： 電晶體此時表現如同一個「恆流源」。電流 ID 主要受閘極電壓控制，幾乎不再隨 VDS 增加而改變。
  • 應用狀態： 這是電晶體作為「訊號放大器」時所在的核心工作區域。
在本程式中的表現：
若設定通道調變參數 (λ) 為 0，則此區為完全水平線；若 λ > 0，則曲線會隨 VDS 增加而微微向上傾斜。

漏電流參數說明 (Leakage Current)

IDSS：汲極漏電流 (Zero Gate Voltage Drain Current)

物理意義： 當閘極接地 (VGS = 0V)，即電晶體處於完全關閉狀態時，從汲極流向源極的微小漏電流。

  • 測試條件： 通常在 VDS 等於最大耐壓 VDSS 時測量。
  • 典型數值： 微安培等級 (μA)。
在本程式中的表現：
由於特性曲線的電流座標通常以「安培 (A)」為單位，IDSS 量級極小，在繪圖時會呈現為貼近 0 的水平線，肉眼難以察覺其變化。

IGSS：閘極漏電流 (Gate-Body Leakage Current)

物理意義： 穿過閘極絕緣層（二氧化矽）的極微小電流，代表閘極控制端的絕緣品質。

  • 測試條件： 在 VDS = 0V，且閘極施加最大工作電壓時測量。
  • 典型數值： 奈安培等級 (nA)。
在本程式中的表現：
IGSS 屬於閘極控制迴路的損耗，並非汲極電流 ID 的一部分，因此完全不會影響特性曲線 (ID vs VDS) 的走勢。
重要提示：
MOSFET 的「飽和區」術語與 BJT 完全相反。在 BJT 中，水平區稱為「主動區」，陡峭區才稱為「飽和區」。請務必區分清楚。

* 註：若 IDSS 或 IGSS 數值異常過大，通常代表元件損毀、靜電擊穿或工作溫度已超出規格限制。
